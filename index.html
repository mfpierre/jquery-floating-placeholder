Let’s Test Some Markup

A very simple and light weight jQuery plugin that makes the placeholder float above the input when typing

Based on a concept from Matt D. Smith on Dribble

Demo

Usage

Include jQuery and jquery-floating-placeholder.js and initialize the plugin like this :

$( 'input' ).floatingPlaceholder();

Sample CSS

You can fully customize your inputs and what the placeholder above with look like, here is the sample CSS used in the demo

input {
    border: 0;
    outline: none;
    margin-top: 5px;
    font: 14px/1.5 Lato, "Helvetica Neue", Helvetica, Arial, sans-serif;
    width: 100%;
}

.placeholder-helper{
    position:absolute;
    font-size: 12px;
    top:5px;
    transition: color .2s ease-in;
}

Options

placeholderActiveColor
The placeholder color when the field is focus
placeholderIdleColor
The placeholder color when the field is not focus
activeClass
The class to use when a placeholder is active
helperClass
The class of the placeholder element
transitionTime
The transition time for the slide effect and color fade-in
